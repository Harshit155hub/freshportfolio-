# Harshit Malhotra — Portfolio Website

A modern, futuristic dark-themed personal portfolio website built with pure HTML, CSS, and JavaScript.

## 🔗 Live Demo
> Deploy via GitHub Pages — see instructions below.

## ✨ Features
- Futuristic dark theme with neon accents (teal, indigo, purple)
- Light / Dark mode toggle
- Smooth scroll-reveal animations
- Fully responsive (mobile + desktop)
- Interactive terminal-style contact card
- Animated project cards with hover effects
- Timeline-style experience section
- Zero dependencies — pure HTML/CSS/JS

## 🚀 Deploy on GitHub Pages (Free Hosting)

1. **Create a new repository** on GitHub
   - Go to https://github.com/new
   - Name it: `portfolio` (or `harshit-portfolio`)
   - Set it to **Public**
   - Click **Create repository**

2. **Upload the files**
   - Click **"uploading an existing file"** or drag & drop `index.html` and `README.md`
   - Commit the files

3. **Enable GitHub Pages**
   - Go to your repo → **Settings** → **Pages**
   - Under **Source**, select `main` branch and `/ (root)` folder
   - Click **Save**

4. **Your site will be live at:**
   ```
   https://<your-github-username>.github.io/portfolio/
   ```

## 🛠️ Customization
- Open `index.html` in any text editor (VS Code recommended)
- Update the resume button link by searching for `alert('Link your resume PDF here!')`  
  and replacing it with a direct link to your hosted PDF
- Add your GitHub profile URL by searching for `https://github.com/` in the contact section
